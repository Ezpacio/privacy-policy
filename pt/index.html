Política de Privacidade

Última atualização: 17.01.2025

Esta política de privacidade descreve nosso compromisso em proteger as informações pessoais de todos os usuários do aplicativo Exercícios para Mulheres. Leia esta política com atenção.

Dados Coletados

Exercícios para Mulheres não coleta nenhuma informação pessoal dos usuários. Todos os dados dentro do aplicativo (por exemplo, número de exercícios, calorias queimadas, dias de exercício) são armazenados apenas no dispositivo do usuário e não são compartilhados com terceiros.

Armazenamento e segurança dos dados

O aplicativo armazena todos os dados localmente em um banco de dados no seu dispositivo. Essas informações são visíveis apenas para o usuário e não são transferidas para nenhum servidor.

Serviços de terceiros

Exercícios para Mulheres não utiliza serviços de terceiros nem ferramentas de análise. Portanto, nenhum dado do usuário é compartilhado com outra plataforma.

Uso de conteúdo

Os guias de exercícios no aplicativo são projetados visualmente para ajudar os usuários a executar corretamente os movimentos. Esses guias são GIFs licenciados e pagos, e são usados apenas para fins informativos e educacionais.

Direitos dos usuários

Como não são coletados dados pessoais, não há necessidade de solicitar acesso, correção, exclusão ou portabilidade dos dados. Se tiver alguma dúvida ou preocupação, não hesite em entrar em contato conosco.

Alterações

Esta política de privacidade pode ser atualizada periodicamente. As atualizações serão anunciadas no aplicativo e/ou nas plataformas relevantes.
