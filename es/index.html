Política de privacidad

Última actualización: 17.01.2025

Esta política de privacidad describe nuestro compromiso de proteger la información personal de todos los usuarios de la aplicación Ejercicios para Mujeres. Lea esta política detenidamente.

Datos recopilados

Ejercicios para Mujeres no recopila ninguna información personal de los usuarios. Todos los datos dentro de la aplicación (por ejemplo, el número de ejercicios, las calorías quemadas, los días de ejercicio) se almacenan únicamente en el dispositivo del usuario y no se comparten con terceros.

Almacenamiento y seguridad de datos

La aplicación almacena todos los datos localmente en una base de datos en su dispositivo. Esta información solo es visible para el usuario y no se transfiere a ningún servidor.

Servicios de terceros

Ejercicios para Mujeres no utiliza servicios de terceros ni herramientas de análisis. Por lo tanto, ningún dato de usuario se comparte con otra plataforma.

Uso de contenido

Las guías de ejercicios en la aplicación están diseñadas visualmente para ayudar a los usuarios a realizar correctamente los movimientos. Estas guías son GIF licenciados y pagados, y se utilizan únicamente con fines informativos y educativos.

Derechos del usuario

Dado que no se recopilan datos personales, no es necesario solicitar acceso, corrección, eliminación o portabilidad de datos. Si tiene alguna pregunta o inquietud, no dude en ponerse en contacto con nosotros.

Cambios

Esta política de privacidad puede actualizarse de vez en cuando. Las actualizaciones se anunciarán dentro de la aplicación y/o en las plataformas relevantes.
